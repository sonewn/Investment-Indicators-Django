사용방법

1. 기업과 날짜 등의 필요 정보를 입력하면 예측을 시작합니다
2. 예측 날짜로부터 예측 기간까지의 종가를 예측합니다
3. 충분한 일수가 있다면 백테스팅을 수행합니다 (예측 기간 세배 이상의 훈련 기간 요구)
4. 사용한 feature 쓰기

사용 가능 모델 (모델이름 ---> "입력문자열")

1. Linear ---> 선형상관 관계를 모델링하는 분석기법
2. ElasticNet ---> 패털티를 선형으로 결합하는 정규화 회귀기법
3. KNN ---> 패턴 인식에서 분류나 회귀에 사용되는 비모수 방식
4. XGBoost(gbtree) ---> Extreme Gradient Boosting, Ensemble 기법 중 하나
5. LightGBM(RandomForest) ---> 배깅 랜덤과정을 추가한 lgbm
6. LightGBM(GOSS) ---> 기울기 기반 단측 표본 추출을 적용한 lgbm
7. ARIMA ---> 자동회귀이동평균 모델의 일반화
8. MLP(LSTM) ---> 이전 단계 정보를 memory cell에 저장하여 다음 단계로 전달하는 모델
